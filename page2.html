3rd Year website

VroomMates

Here we created a website to help tackle the challenge of students.

The website set out the task of allowing students to login, set their travel route and by doing so it allowed other members to join. If you seen a person could pick you up along a route you are allowed communicate via phone. We set up routine membership data such as gender, location etc. Each person offering the lift would come to an agreement with the passenger and benefit both parties. All data was store on firebase for login credintials and matchmaking.

Actionscript browser game

Searchlight , Adobe flash

This team project we based it on a search and find game

The games main quest was to locate a four digit code from 4 rooms. These four rooms contained one digit for the code. Once all digits were located a number pad was revealed and the player could exit the game. Each team memeber had to make their own level and hide the relevant number.
I was in charge of pulling it all together also so I set the whole combination screen and introduction screen.

Ionic Mobile application

MaynoothPool

The applcation was created to allow students to travel with each other both saving money and supporting a greener campus

This mobile application was completed as my part of my final year thesis and done in Ionic 4 with AngularJS and TypeScript. It aimed to achieve a greener campus and also acted as a secondary taxi carpooling app in the interest of saftey and utilizing all aspects of the application. This is all too tackle the ever growing traffic pollution we see everyday on our way to work etc. The problem of one person cars traveling long distances.
Several features included:

Application Login/Authentication ∇

Connecting and utilizing Firebase and user profiles ∇

Allow users to interact with maps , routes and geolocations ∇

Calculating distance between google locations ∇

Google maps API ∇

If I had to critise the app I would say that it could have been much bigger but it also leaves a lot of growth for the application.
